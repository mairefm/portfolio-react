# Portfólio Pessoal — React

Este é o meu portfólio desenvolvido em **React**, criado com foco em estudos de componentes, organização de projeto e construção de interfaces modernas.  
Ele funciona como uma **demonstração visual** das minhas habilidades e projetos, além de complementar meu portfólio principal em **Angular**, que é minha stack foco no momento.

---

## Objetivo do Projeto

- Criar um portfólio moderno e responsivo  
- Testar componentização em React  
- Organizar seções de forma clara: Sobre, Skills, Projetos e Contato  
- Consolidar meus estudos de frontend  
- Exibir meus projetos do GitHub de maneira visual  

---

## Tecnologias Utilizadas

- **React**
- **JavaScript (ES6+)**
- **CSS / Design Responsivo**
- **Create React App**
- **React Hooks**

---

## Como rodar o projeto localmente
```bash
# Clonar o repositório
git clone https://github.com/mairefm/portfolio-react.git

cd portfolio-react

# Instalar as dependências
npm install

# Rodar o servidor de desenvolvimento
npm start

# Acessar no navegador
http://localhost:3000
